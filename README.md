# Welcome to PopTag! 🎈
Welcome to the PopTag Installer!
PopTag! (also known as Crazy Arcade) was a free-to-play, 2D browser-based online arcade game published by Nexon, released in 2010 as the North American version of "Crazy Arcade BnB". It originially shut down in July 2011, due to "multiple factors including the inability to deliver new content regularly due to the limited resources available for the game".
## Installation Steps
1. Manually unzipping and installing
   You can choose to download this repository via a .zip file and run CA.exe.
2. PopTag setup wizard
Just navigate over to the Releases page found on the side of the page and run the Setup Wizard.
